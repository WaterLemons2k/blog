  2. 转到计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\W32Time\TimeProviders\NtpServer,将Enabled的值设置为1，如下图所示:
  3. 转到计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\W32Time\Config,将AnnounceFlags的值设置为5，如下图所示:
