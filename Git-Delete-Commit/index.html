Git删除Commit

2022-03-22
  1. 输入以下命令来查看历史提交的commit
git log
  1. 找到要删除的commit的注释，记住下一行的commit号(如图中红框)
  2. 然后执行
git rebase -i commit号


4. 将要删除的commit号前面的pick替换为drop
5. 输入以下命令检查是否删除

git log
  1. 最后输入以下命令推送至仓库即可
git push origin HEAD --force
